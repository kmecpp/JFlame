# JFlame
A lightweight, yet extremely fast JSON parser and generator

Examples:

Parse a JSON string

```
JsonValue value = Json.parse(jsonString);
```

## Maven
```
	<repositories>
		<repository>
			<id>kmecpp-repo</id>
			<url>http://repo.kmecpp.com/maven</url>
		</repository>
	</repositories>

	<dependencies>
		<dependency>
			<groupId>com.kmecpp</groupId>
			<artifactId>jflame</artifactId>
			<version>1.0</version>
		</dependency>
	</dependencies>
	```
